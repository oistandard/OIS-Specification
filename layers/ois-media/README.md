# OIS-Media

## Scope

Media lifecycle, ingestion, and delivery across signage networks.

## Provider to Service (Ingest)

* Not defined for OIS-Media.

## Service to Provider (Delivery)

* Deliver media packages and playback instructions.
* Specify schedules, triggers, and caching requirements.
