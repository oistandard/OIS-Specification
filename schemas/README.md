# Schemas

This directory contains JSON Schema files that define the canonical types for
OIS payloads. Examples in `examples/` should validate against these schemas.

Layer schemas:

* `schemas/ois-core/authentication.schema.json`
* `schemas/ois-display/screen-inventory.schema.json`
* `schemas/ois-media/media-delivery.schema.json`
* `schemas/ois-proof-of-play/playback-event.schema.json`
* `schemas/ois-sense/sensor-event.schema.json`
* `schemas/ois-location/store-layout.schema.json`
* `schemas/ois-events/realtime-event.schema.json`
* `schemas/ois-commerce/transaction-attribution.schema.json`
